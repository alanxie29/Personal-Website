SubTracker

Summary: SubTracker is an application that allows you to keep track of all your subscriptions. Using a mobile application, users can authenticate, add, and edit their subscriptions and record their cost and renewal period.

Technology: Ionic 4, Angular 7, MongoDB, Node.js (Express.js, Passport.js), Mongoose, HTML/CSS, TypeScript, Postman

My Tasks:

  • Created User entities and nested Subscriptions entities to be saved in MongoDB database.

  • Utilized Passport.js and Ionic local storage to implement authentication using JSON Web Tokens.

  • Configured Mongoose to simplify interactions with MongoDB database.

  • Wrote a RESTful API using Node.js and Mongoose to handle authentication and CRUD operations within application endpoints.

  • Used Angular form logic and HTTP Module to call REST API endpoints

  • Developed an Ionic 4 client in TypeScript to send HTTP requests and receive data from Node.js backend.

    • Purpose

    SubTracker served as a way for me to learn about HTTP requests and backend operations, specifically with Node.js and Angular 7. The application also documents my learning experience with saving documents and nested documents within MongoDB using Node.js and Mongoose. Using these tools, I created a fully functioning Ionic 4 client end and a Node.js server to handle all processes and requests.

    Next Steps

    A major improvement to the app could be made with SMS or push notification features using the Twilio API or the Ionic API to remind users of their upcoming subscription renewal date. Another improvement could feature links to each subscription's account to provide the user a direct link to cancel or renew their subscription.
